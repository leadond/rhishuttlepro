# Shuttle Pro

Shuttle Pro is a comprehensive transportation management system designed for hotels and shuttle services.

## Features

- **Dispatcher Control:** Real-time fleet management and ride assignment.
- **Driver Dashboard:** Mobile-friendly interface for drivers to manage rides and vehicle status.
- **Guest Portal:** Easy-to-use booking interface for hotel guests.
- **Public Tracking:** Real-time shuttle tracking for guests.
- **TV Monitor:** Lobby display for upcoming departures.
- **Admin Panel:** User, vehicle, and system management.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Firebase (Firestore, Auth), Vercel Serverless Functions

## Running the App

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Building for Production

```bash
npm run build
```

## Deployment

The application is configured for deployment on Vercel.
